# 在线简历生成器

访问 [在线简历生成器](https://cv.aixia.wang/) 在线体验

## 📦 项目简介
这是一个功能强大的在线简历生成器，支持多种精美模板和实时预览。

## ✨ 功能特性
- 🎨 4种精美简历模板（现代、经典、创意、专业）
- 🌈 6种主题颜色可选
- 📱 完全响应式设计
- 💾 一键导出PDF
- 🔄 实时预览
- 📝 完整的简历数据填写
 

## 📂 文件结构
```
简历/
├── index.html          # 主HTML文件
├── styles.css          # 样式文件
├── app.js             # 应用逻辑
└── README.md          # 本文件
```
 
## 📝 使用说明
1. 打开网站
2. 在左侧填写个人信息
3. 实时预览右侧简历效果
4. 切换模板和颜色主题
5. 点击「下载PDF」导出简历

## 🎯 浏览器支持
- Chrome (推荐)
- Firefox
- Safari
- Edge
- 其他现代浏览器
